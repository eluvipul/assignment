Step 1: Connect via Power Query
Use Data → Get Data → From SQL Server Database to connect and load the Restaurants table into Excel (as you already did earlier).

This connection is now a live link between Excel and SQL Server.

🛠 Step 2: Configure Refresh Settings
Go to the Data tab in Excel.

Click the dropdown under Refresh All → Connection Properties.

In the Usage tab, enable:

Refresh data when opening the file → ensures the latest records load each time you open Excel.

Refresh every X minutes → set an interval (e.g., 10 minutes) for automatic refresh.

Or leave it manual if you prefer one-click refresh.

🛠 Step 3: One-Click Refresh
Once configured, you can simply click Refresh All in the Data tab whenever new records are added in SQL Server.

Excel will query the database again and update the worksheet with the latest rows.
